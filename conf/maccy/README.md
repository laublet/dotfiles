# Maccy Configuration

Clipboard manager pour macOS.

## Installation

```bash
brew install --cask maccy
```

## Restaurer la config

```bash
defaults import org.p0deje.Maccy ~/dev/perso/dotfiles/conf/maccy/maccy.plist
```

## Raccourci

- **Cmd+Option+C** : Ouvrir l'historique

## Config

- Paste automatique après sélection
- Apps ignorées : 1Password, Bitwarden, KeePassXC, LastPass, Keychain Access
